## 🌐 Enterprise-Grade Accessibility Toolkit
The **ATL Productivity Suite** is a flagship, premium-tier global plugin engineered specifically for NVDA screen reader users. Designed for visually impaired professionals, developers, and power users, this Version 2.0 major release transforms NVDA into the ultimate high-performance workstation assistant. By centralizing 10 highly optimized system utilities into a single environment, it bypasses complex Windows visual hierarchies and puts enterprise security scanning, workflow automation, and system diagnostics directly at your fingertips.

---

## 🚀 What's New in Version 2.0.0 (Major Architecture & Feature Update)
Version 2.0 is a massive leap forward. We have removed redundant tools, fixed core bugs, and introduced cutting-edge automation capabilities to make your workflow faster and smarter.

* **Smart Search Hub V2.0:** Upgraded from a simple Google search to a Universal Search Hub. It now supports **10 search engines** (Google, YouTube, Wikipedia, Amazon, Stack Overflow, GitHub, Bing, DuckDuckGo, Reddit, Twitter). Added smart clipboard auto-fill and prefix-based routing (e.g., type `yt: song` to search directly on YouTube).
* **Auto-Cleanup Subsystem:** The System Cleaner now features an automated scheduling engine. Set it to run Daily, Weekly, or Monthly. It also includes new logic to clear the Windows 'Run' dialog history (RunMRU).
* **Voice-Assisted (TTS) Alarms:** The Alarm Manager now supports Text-to-Speech (TTS) integration. Along with custom MP3 ringtones, NVDA will explicitly announce your custom alert messages (e.g., "Time to take your medication") when the alarm rings.
* **Smart Workspace Launcher:** Completely overhauled the Quick Launcher. A single shortcut slot can now act as a "Workspace"—launching multiple applications, folder paths, web URLs, and deep Windows Settings (like Bluetooth or Wi-Fi) simultaneously.
* **Pomodoro Focus Mode:** Added a dedicated Pomodoro timer to the Precision Stopwatch (25 minutes of work followed by a 5-minute break) with audio cues, designed specifically to boost productivity for coders and students.
* **Universal Sentinel Scanner:** The VirusTotal integration has been upgraded. You can now scan URLs, Domains, and IP addresses. Additionally, it now supports multi-part enterprise routing to seamlessly handle and upload large files over 32MB.
* **Advanced Power Warnings:** The Power Manager now features a pre-action audio warning system. A custom alert will play a few minutes before any scheduled Shutdown, Restart, or Sleep event, giving you time to save your work.
* **Deep Diagnostic Scans:** The System Health module now offers two modes: a Standard Scan and an Admin-level Deep Diagnostic Scan for extracting advanced hardware telemetry.

## ⚙️ Core Features & Capabilities (The 10 Utilities)

### 1. ATL Sentinel Security Scanner
Verify the safety of any file (up to 650 MB), domain, URL, or IP address using enterprise-grade threat intelligence. It cross-checks your target against 70+ top-tier antivirus engines and delivers a clean, easily readable diagnostic report confirming whether the target is Malicious, Suspicious, or Safe.

### 2. ATL System Dashboard (Health Monitor)
Gain immediate auditory insight into your computer’s real-time performance. Run Deep Scans to receive precise diagnostic readouts of your CPU load, RAM availability, motherboard details, and precise battery wear/degradation percentages.

### 3. ATL System Cleaner V2.0
Digital clutter slows down screen reader response times. This robust utility automates the safe removal of Windows Temp files, Prefetch data, RunMRU history, and browser caches. With the new **Auto-Cleanup** feature, your system stays optimized automatically in the background.

### 4. Smart Workspace Launcher (10 Slots)
Map up to 10 "Workspaces" to dedicated hotkeys (`NVDA+Windows+0-9`). A single workspace can instantly open your code editor, a specific project folder, your favorite website, and your Windows Bluetooth settings all at once, bypassing the Start Menu entirely.

### 5. ATL Power Manager
Securely execute or schedule Restart, Shutdown, and Sleep commands. Features a strict 5-minute safety cooldown to prevent accidental system crashes and a mandatory audio warning countdown before any power action is executed.

### 6. ATL Alarm Manager
A precision scheduling utility with custom MP3 alerts, snooze functionality, and newly integrated NVDA Voice Assistant (TTS) messages that announce your custom reminders exactly when you need them.

### 7. ATL Smart Search Hub
Trigger a web search from anywhere in your OS. Type your query or use smart prefixes (like `a:` for Amazon or `gh:` for GitHub). The hub reads your clipboard automatically, eliminating the multi-step process of manually opening a browser.

### 8. ATL Precision Stopwatch
A flawless, highly accurate stopwatch for precise time-tracking. Use the Standard Mode for regular task monitoring with Lap recording, or switch to the Pomodoro Mode for structured focus sessions.

### 9. ATL Smart Weather Assistant
Get instant, highly accurate weather reports for your default city or search globally. Supports full bilingual output (English and Hindi), translating complex WMO weather codes into human-readable conditions.

### 10. Unified Master Menu
A centralized, alphabetically sorted hub (`NVDA+Shift+M`) that houses all the utilities. If you forget a specific shortcut key, simply invoke the Master Command to access any tool instantly.

* **About ATL Suite & Help:** `NVDA + Shift + O`
* **Power Manager:** `NVDA + Shift + P`
* **Smart Workspace Config:** `NVDA + Shift + Q`
* **Smart Weather Assistant:** `NVDA + Shift + W`

### Smart Workspace Execution
* **Execute Slots 1 to 9:** `NVDA + Windows + 1 to 9`
* **Execute Slot 10 (0):** `NVDA + Windows + 0`

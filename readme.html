ATL Productivity Suite v2.0

The Ultimate Enterprise-Grade Accessibility Toolkit for NVDA

Overview

The ATL Productivity Suite is a flagship, premium-tier global plugin engineered specifically by Accessible Tech Labs India for NVDA screen reader users. Designed for visually impaired professionals, developers, and power users, this Version 2.0 major release transforms NVDA into a high-performance workstation assistant. By centralizing 10 highly optimized system utilities into a single environment, it bypasses complex Windows visual hierarchies and puts enterprise security scanning, workflow automation, and deep system diagnostics directly at your fingertips.

What's New in Version 2.0.0 (Major Architecture & Feature Update)

Version 2.0 is a massive leap forward. We have removed redundant tools, fixed core bugs, and introduced cutting-edge automation capabilities to make your workflow faster and smarter.

1. Smart Search Hub V2.0

Upgraded from a simple Google search to a Universal Search Hub. It now supports 10 search engines (Google, YouTube, Wikipedia, Amazon, Stack Overflow, GitHub, Bing, DuckDuckGo, Reddit, Twitter). We also added smart clipboard auto-fill and prefix-based routing (e.g., type yt: song to search directly on YouTube).

2. Auto-Cleanup Subsystem

The System Cleaner now features an automated scheduling engine. Set it to run Daily, Weekly, or Monthly in the background. It also includes new logic to deep-clean the Windows 'Run' dialog history (RunMRU).

3. Voice-Assisted (TTS) Alarms

The Alarm Manager now supports NVDA Text-to-Speech (TTS) integration. Along with custom MP3 ringtones, NVDA will now explicitly announce your custom alert messages (e.g., "Time to take your medication") exactly when the alarm rings.

4. Smart Workspace Launcher

We completely overhauled the Quick Launcher. A single shortcut slot can now act as a "Workspace"—launching multiple applications, folder paths, web URLs, and deep Windows Settings (like Bluetooth, Wi-Fi, or Sound) all simultaneously with one keystroke.

5. Pomodoro Focus Mode

Added a dedicated Pomodoro timer to the Precision Stopwatch (25 minutes of work followed by a 5-minute break). It features audio cues and is designed specifically to boost productivity for coders, students, and professionals.

6. Universal Sentinel Scanner

The VirusTotal integration has been significantly upgraded. You can now scan URLs, Domains, and IP addresses. Additionally, it now supports multi-part enterprise routing to seamlessly handle and upload large files over 32MB.

7. Advanced Power Warnings

The Power Manager now features a pre-action audio warning system. A custom alert will play a few minutes before any scheduled Shutdown, Restart, or Sleep event, giving you ample time to save your critical work.

8. Deep Diagnostic Scans

The System Health module now offers two specific modes: a fast Standard Scan, and an Admin-level Deep Diagnostic Scan for extracting advanced hardware telemetry and motherboard details.

Core Features & Capabilities (The 10 Utilities)

  • ATL Sentinel Security Scanner: Verify the safety of any file (up to 650 MB), domain, URL, or IP address using 70+ top-tier antivirus engines.
  • ATL System Dashboard: Run Deep Scans to receive precise diagnostic readouts of your CPU load, RAM availability, and precise battery wear/degradation percentages.
  • ATL System Cleaner V2.0: Automates the safe removal of Windows Temp files, Prefetch data, RunMRU history, and browser caches with the new Auto-Cleanup scheduler.
  • Smart Workspace Launcher (10 Slots): Map up to 10 "Workspaces" to dedicated hotkeys. Open your code editor, a project folder, and a website all at once.
  • ATL Power Manager: Securely schedule Restart, Shutdown, and Sleep commands with a mandatory audio warning countdown before execution.
  • ATL Alarm Manager: Precision scheduling with custom MP3 alerts, snooze functionality, and integrated NVDA Voice Assistant (TTS) messages.
  • ATL Smart Search Hub: Search across 10 global engines directly from your OS. The hub reads your clipboard automatically to save typing time.
  • ATL Precision Stopwatch: A highly accurate stopwatch featuring both Standard Mode (with Lap recording) and Pomodoro Focus Mode.
  • ATL Smart Weather Assistant: Get instant, highly accurate weather reports for your default city or globally, with full bilingual output (English and Hindi).
  • Unified Master Menu: A centralized hub housing all the utilities. If you forget a specific shortcut, simply invoke the Master Command to access any tool instantly.

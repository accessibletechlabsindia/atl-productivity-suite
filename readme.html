ATL Productivity Suite v1.5

The Enterprise-Grade Accessibility Toolkit for NVDA

Overview

The ATL Productivity Suite is a premium, all-in-one efficiency toolkit engineered by Accessible Tech Labs India. Specifically designed for blind and visually impaired professionals, it seamlessly integrates essential system utilities directly into the NVDA screen reader environment. Version 1.5 introduces a high-speed unified master database layout, ensuring unparalleled stability, advanced security, intelligent resource management, and strict privacy compliance.

What's New in Version 1.5 (Performance & Architecture Update)

Version 1.5 focuses on core engine optimization and complete compatibility with the latest NVDA security patches, alongside our existing suite of premium features.

1. Unified Master Database Architecture

We have completely restructured the backend configuration system. By migrating multiple disparate settings files into a single, high-speed unified master JSON database, the addon now loads instantly, consumes significantly less memory, and executes tasks with zero latency.

2. Verified NVDA 2026.1.1 Compatibility

Ensured seamless integration and absolute stability with the latest NVDA 2026.1.1 security patches, guaranteeing an uninterrupted and secure user experience.

3. ATL Sentinel Security Scanner

Visually impaired users often struggle to navigate complex and cluttered antivirus interfaces. To solve this, we have integrated an enterprise-grade security module powered by the industry-leading VirusTotal API. You can now instantly scan any local file (up to 650 MB) or suspicious web URL against more than 70 top-tier antivirus engines.

4. ATL Precision Stopwatch

A flawlessly accurate, fully accessible stopwatch utility tailored for professionals and students. Whether you need to track the time taken to compile a piece of code or monitor daily tasks, it announces the elapsed time clearly, making time-tracking entirely independent.

5. Intelligent Telemetry & Strict Privacy Compliance

To continually improve the ATL Suite, we have implemented a smart, real-time analytics system. Your privacy is our absolute priority. This system operates strictly on an opt-in basis. We guarantee that NO personal data, NO file names, NO PC identifiers, and NO browsing history are ever collected.

6. Advanced System Cleaner Enhancements

The tool intelligently requests Administrative privileges when deep-cleaning system directories. This puts the power entirely in the user's hands—allowing for safe, user-authorized deep cleaning of Windows Temp folders, Prefetch data, and browser caches.

7. Power Manager Safety Protocols

We have introduced a strict "Cooldown Protocol" to the Power Manager. The system now enforces a mandatory 5-minute safety gap between scheduled tasks, providing clear, professional audio warnings if a conflict is detected.

Core Utilities & Master Menu

  • System Health Dashboard: Real-time monitoring of CPU processing load, RAM availability, and precise battery status.
  • Advanced System Cleaner: Deep-cleans system clutter with robust, user-authorized administrative execution.
  • Smart Quick Launcher: Map up to 10 favorite applications, folders, or web links to dedicated hotkeys (NVDA+Windows+0-9) for instantaneous workflow acceleration.
  • Alarm Manager: Precision scheduling utility with high-fidelity custom audio alerts.
  • Power Manager: Securely execute Shutdown, Restart, and Sleep commands with advanced crash-prevention logic.
  • Google Quick Search: Instant web search execution without manually opening a browser.
  • ATL Sentinel Security Scanner: Scan local files and web URLs against 70+ antivirus engines.
  • ATL Precision Stopwatch: Accessible time-tracking for any task.
  • Unified Master Menu: A centralized, alphabetically sorted hub to access all tools seamlessly without memorizing individual shortcuts.

Installation & Support

To install, simply run the atl_productivity_suite_1.5.0.nvda-addon file and follow the standard NVDA installation prompts. The tool requires a minimum NVDA version of 2021.1 and has been fully tested on the latest NVDA 2026.1.1 releases.

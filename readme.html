ATL Productivity Suite v1.4

The Enterprise-Grade Accessibility Toolkit for NVDA

Overview

The ATL Productivity Suite is a premium, all-in-one efficiency toolkit engineered by Accessible Tech Labs India. Specifically designed for blind and visually impaired professionals, it seamlessly integrates essential system utilities directly into the NVDA screen reader environment. Version 1.4 is a major architectural overhaul, transforming the addon from a basic utility script into a high-performance, enterprise-grade modular application. This update focuses on unparalleled stability, advanced security, intelligent resource management, and strict privacy compliance.

What's New in Version 1.4 (Massive Update)

Version 1.4 is our biggest release to date, bringing critical architectural improvements, entirely new enterprise-level features, and crucial bug fixes based on community feedback. Below is an in-depth breakdown of all the modifications.

1. Brand New Feature: ATL Sentinel Security Scanner

Visually impaired users often struggle to navigate complex and cluttered antivirus interfaces. To solve this, we have integrated an enterprise-grade security module powered by the industry-leading VirusTotal API. You can now instantly scan any local file (up to 650 MB) or suspicious web URL against more than 70 top-tier antivirus engines. It provides a clean, highly accessible, and easy-to-read safety report directly within NVDA, ensuring your system remains entirely secure from malware, ransomware, and phishing threats.

2. Brand New Feature: ATL Precision Stopwatch

A flawlessly accurate, fully accessible stopwatch utility tailored for professionals and students. Whether you need to track the time taken to compile a piece of code, time a mock examination, or monitor daily tasks, this tool provides seamless start, stop, and reading functionalities. It announces the elapsed time clearly, making time-tracking entirely independent, hassle-free, and accessible without visual reliance.

3. Intelligent Telemetry & Strict Privacy Compliance

To continually improve the ATL Suite, we have implemented a smart, real-time analytics system. Your privacy is our absolute priority. This system operates strictly on an opt-in basis; you will be prompted with a secure Yes/No dialog upon first execution. We guarantee that NO personal data, NO file names, NO PC identifiers, and NO browsing history are ever collected. We solely track anonymous usage statistics (e.g., which tool was launched) to help us prioritize feature development. Furthermore, the system is equipped with an offline-sync mechanism, ensuring that if you are disconnected from the internet, data is securely cached locally and synced seamlessly upon reconnection.

4. Modular Code Architecture Restructuring

In previous versions, core functionalities such as the Quick Launcher, System Cleaner, and Quick Search were all bundled within a single, massive core file. This led to performance bottlenecks and difficult maintenance. In Version 1.4, we have completely refactored the codebase. Every single utility now operates within its own dedicated, isolated Python module. The main plugin file now acts solely as a central dispatcher. This enterprise-level modularization drastically reduces memory consumption, enhances execution speed, prevents cross-module interference, and paves the way for seamless future updates.

5. Advanced System Cleaner Enhancements

The System Cleaner has received a critical logic upgrade. Previously, the cleaner would execute regardless of the NVDA privilege level, which could lead to permission errors. The tool now intelligently requests Administrative privileges when deep-cleaning system directories. This puts the power entirely in the user's hands—allowing for safe, user-authorized deep cleaning of Windows Temp folders, Prefetch data, and browser caches, ensuring maximum hygiene without risking system stability.

6. Power Manager Safety Protocols & Audio Feedback

We have introduced a strict "Cooldown Protocol" to the Power Manager. Previously, users could accidentally schedule multiple conflicting power tasks (e.g., Shutdown and Restart) within a five-minute window, potentially causing catastrophic system or NVDA crashes. The system now enforces a mandatory 5-minute safety gap between scheduled tasks, providing clear, professional audio warnings if a conflict is detected. Additionally, we have integrated a high-fidelity, custom notification tone. Whenever a power task executes or a notification is triggered, you will now hear a premium audio cue, providing a true native-system feel.

7. Alarm Manager Bug Fixes

We have resolved several underlying bugs within the Alarm Manager logic. The scheduling algorithm has been refined to ensure that alarms trigger with exact precision, and issues related to alarm deletion and modification have been completely eradicated, providing a flawless time-management experience.

8. Revamped About Manager & Help Documentation

The 'About' section has been entirely redesigned. It now includes comprehensive, 500+ character explanations for every feature, an integrated link to our official step-by-step YouTube video tutorial, and a dedicated section detailing every single shortcut and gesture. Furthermore, we have expanded our support channels, adding direct, accessible links to contact the developer via WhatsApp, Telegram, Outlook, and Gmail directly from the addon menu.

9. Deprecation Notice: Text File Reader Removed

Based on our user analytics and community feedback, the 'Accessible File Reader' feature has been entirely removed from the suite. The tool was deemed redundant due to the native capabilities of modern text editors like Notepad and Word. Removing it has further reduced the addon's footprint, making the entire suite significantly faster and leaner.

Core Utilities & Master Menu

  • System Health Dashboard: Real-time monitoring of CPU processing load, RAM availability, and precise battery status.
  • Advanced System Cleaner: Deep-cleans system clutter with robust, user-authorized administrative execution.
  • Smart Quick Launcher: Map up to 10 favorite applications, folders, or web links to dedicated hotkeys (NVDA+Windows+0-9) for instantaneous workflow acceleration.
  • Alarm Manager: Precision scheduling utility with high-fidelity custom audio alerts.
  • Power Manager: Securely execute Shutdown, Restart, and Sleep commands with advanced crash-prevention logic.
  • Google Quick Search: Instant web search execution without manually opening a browser.
  • ATL Sentinel Security Scanner: Scan local files and web URLs against 70+ antivirus engines.
  • ATL Precision Stopwatch: Accessible time-tracking for any task.
  • Unified Master Menu: A centralized, alphabetically sorted hub to access all tools seamlessly without memorizing individual shortcuts.

Installation & Support

To install, simply run the atl_productivity_suite_1.4.0.nvda-addon file and follow the standard NVDA installation prompts. The tool requires a minimum NVDA version of 2021.1 and has been fully tested on the latest NVDA 2026.1 releases.
